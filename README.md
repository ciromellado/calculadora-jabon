# Calculadora Jabonilandia 🧼

Calculadora web profesional diseñada para la elaboración de jabones artesanales. Esta aplicación permite realizar cálculos precisos de saponificación, gestionar sobreengrasado y calcular cantidades de ingredientes para lotes de jabón, incluyendo un protocolo para extractos alcohólicos.

## Características principales ✨

- **Cálculo automático:** Determinación precisa de Sosa (NaOH) y agua según la formulación de aceites.
- **Protocolo de Extractos:** Cálculo integrado para la adición de extractos alcohólicos (máximo permitido del 2.5%).
- **Control de Calidad:** Cálculo del Índice de Yodo para verificar la estabilidad de la mezcla y prevenir la oxidación.
- **Experiencia Offline (PWA):** Instalable en escritorio y dispositivos móviles para acceder a la herramienta sin conexión a internet.
- **Interfaz Intuitiva:** Diseño limpio y fácil de usar, enfocado en el flujo de trabajo del artesano.

## Tecnologías utilizadas 🛠️

- HTML5
- CSS3 (Diseño responsivo)
- JavaScript (Lógica de cálculo)
- PWA (Service Workers y Manifest para instalación)

## ¿Cómo instalar? 📱

1. Abre esta página en tu navegador (Chrome, Edge o Safari).
2. Busca el icono de "Instalar" (un monitor con una flecha hacia abajo) en la barra de direcciones.
3. Haz clic e instala. La calculadora aparecerá en tu escritorio o pantalla de inicio como una aplicación nativa.
